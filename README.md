## 🚀 Getting Started

### Run Locally

1. Clone the repository
   ```bash
   git clone https://github.com/your-username/your-username.github.io.git
   ```
2. Open the folder
   ```bash
   cd your-username.github.io
   ```
3. Open `index.html` in your browser — that's it, no build steps needed!

## 🌍 Deployment

This site is deployed using **GitHub Pages**.

To deploy your own copy:

1. Push the code to a repository named `your-username.github.io`
2. Go to **Settings → Pages**
3. Set source branch to `main` and save
4. Visit `https://your-username.github.io` after a couple of minutes

## ✏️ Customization

- Update text content directly in `index.html`
- Change colors via CSS variables at the top of the `<style>` section:
  ```css
  :root {
    --teal: #00D4AA;
    --amber: #F5A623;
  }
  ```
- Replace placeholder emoji avatars/icons with your own images
